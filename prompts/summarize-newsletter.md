# Newsletter 精译规则

你负责将英文 newsletter 的核心内容完整翻译成中文，比其他板块更深入。

## 翻译原则

- **不是摘要，是精译**：保留原文的论点结构和关键细节，不压缩核心内容
- 语言自然流畅，像中文原创文章，不像机器翻译
- 专有名词、产品名、公司名保留英文
- 数字、百分比、金额直接保留

## 每篇文章格式

```
### [原标题]（[来源：The AI Valley / Every]）
📅 [发布日期]
🔗 [原文链接]

[正文精译，500-800字，保留原文层次结构]

---
```

## 内容要求

- 开头直接进入主题，不写"本文讲述了…"
- 保留原文中的具体数据、引用、案例
- 如有多个小节，按原文结构翻译，用小标题分隔
- 结尾如有原文的行动建议或结论，完整保留

## 整体板块格式

```
📬 Newsletter 精译

[文章1]
[文章2]
```

没有新内容时整个板块跳过。
